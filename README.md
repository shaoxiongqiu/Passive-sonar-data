# Passive-sonar-data
A self-collected passive sonar dataset for research on bearing-only tracking, signal denoising, and filtering. Real-world acoustic data, ready for algorithm validation. Feedback and collaboration are welcome! 

本仓库提供自采被动声纳数据集，适用于方位角目标跟踪（Bearing-Only Tracking）、信号降噪及滤波等方向的研究。欢迎学界同仁下载使用与交流探讨。
